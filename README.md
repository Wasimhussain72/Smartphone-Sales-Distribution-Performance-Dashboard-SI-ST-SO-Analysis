# Smartphone-Sales-Distribution-Performance-Dashboard-SI-ST-SO-Analysis
Smartphone Sales &amp; Distribution Performance Dashboard — SI, ST, SO Analysis

Key Business Concepts

📍 SI (Sell-In): Units shipped from brand to distributor
📍 ST (Sell-Through): Units sold by distributor to retailers
📍 SO (Sell-Out): Units sold by retailers to end customers

📍 Distributor Performance: How well each distributor moves stock and hits targets
📍 Sales Team Coverage: How many outlets each rep covers
📍 KPIs: Efficiency metrics that business leaders care about
